# Indexer Control Panel (Localhost)
Modern localhost GUI for folder selection + live dashboard for GPU/CPU/RAM + indexing stats.

## Quickstart
1) Backend:
```bash
cd apps/backend
npm install
npm run dev
```

2) Frontend:
```bash
cd apps/frontend
npm install
npm run dev
```

- Frontend: http://127.0.0.1:5173  
- Backend:   http://127.0.0.1:8787
